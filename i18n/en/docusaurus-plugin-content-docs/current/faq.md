# Frequently Asked Questions

## Hardware Related

### 1. Servo not moving?

- Check if servo wiring is correct
- Confirm power supply is normal
- Verify firmware is flashed correctly
- Check if servo is damaged

### 2. Main board not recognized?

- Check USB connection
- Confirm driver is installed correctly
- Try replacing USB cable
- Check if main board is damaged

### 3. Sensor not responding?

- Check if wiring is correct
- Confirm sensor model matches
- Verify program configuration is correct
- Check if sensor is damaged

## Software Related

### 1. Firmware flashing failed?

- Check USB connection
- Confirm flashing tool version
- Verify firmware file integrity
- Try reducing baud rate

### 2. AI functions not working?

- Check network connection
- Confirm API key configuration
- Verify model is loaded correctly
- Check server status

### 3. System running unstably?

- Check memory usage
- Confirm firmware version
- Verify configuration parameters
- Clear system cache

## Assembly Related

### 1. Structure unstable?

- Check if screws are tightened
- Confirm 3D printed parts are not deformed
- Adjust center of gravity
- Check if connections are secure

### 2. Movement not smooth?

- Check servo installation position
- Confirm mechanical structure is not stuck
- Adjust movement parameters
- Check power supply

### 3. Difficulty installing shell?

- Check printed part dimensions
- Confirm installation order
- Adjust fit appropriately
- Use appropriate tools

## Network Related

### 1. Cannot connect to Wi-Fi?

- Check network settings
- Confirm password is correct
- Verify network compatibility
- Try restarting device

### 2. Remote control not responding?

- Check network connection
- Confirm APP version
- Verify device online status
- Check firewall settings

## Other Issues

### 1. Short battery life?

- Check battery capacity
- Confirm charging is normal
- Optimize power consumption settings
- Consider replacing battery

### 2. Audio output abnormal?

- Check speaker connection
- Confirm volume settings
- Verify audio files
- Check if hardware is damaged
