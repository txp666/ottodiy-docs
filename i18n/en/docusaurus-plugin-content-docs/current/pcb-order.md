## PCB Ordering

1. Visit [LCEDA Open Source Project Link](https://oshwhub.com/txp666/ottorobot)
2. Log in to your LCEDA open source account (register first if you don't have one)
3. Click "One-Click Order PCB" in LCSC Mall
